# Getting Started

ProteinClaw is a conversational agent that queries protein databases using a ReAct reasoning loop. This guide gets you running in under 5 minutes.

## Download

Download the macOS app from the [latest release](https://github.com/shuaizengMU/ProteinClaw/releases/latest/download/ProteinClaw.dmg).

On first launch, macOS may warn about an unidentified developer. Open **System Settings → Privacy & Security**, scroll to the bottom, and click **Open Anyway**.

## Your first query

Once launched, type a research question in plain language. For example:

> What are the pathogenic ClinVar variants of BRCA1, and which ones fall in the RING domain?

ProteinClaw will:

1. **Think** — plan which databases to query.
2. **Act** — call UniProt, ClinVar, and PDB in sequence.
3. **Answer** — synthesize a cross-referenced answer with a reproducible trace.
